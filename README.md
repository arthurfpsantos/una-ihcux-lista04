O try-catch é uma estrutura de controle fundamental na programação utilizada para o tratamento de exceções (erros que ocorrem durante a execução do programa). Ele permite que um programa lide com situações inesperadas de forma graciosa, em vez de travar ou fechar inesperadamente.
